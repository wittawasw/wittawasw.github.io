จัดรูปแบบของ paragraph ให้เหมือนหนังสือด้วย text-justify

Mar 29, 2024

เร็วๆนี้มีเหตุให้ต้องทำการ render HTML เพื่อแปลงเป็น PDF สำหรับใช้เป็นเอกสารทางการ ก็เลยได้วิธีจัดย่อหน้ามาใช้ด้วย ถ้าต้องทำรูปแบบเหมือนจดหมายทางการ

วิธีการก็คือใช้ CSS แบบนี้

p.indented-full {
  /* เว้นช่องว่างในบรรทัดแรกของ paragraph */
  text-indent: 30px;
  /* ขยายช่องว่างระหว่างคำให้เต็มบรรทัด */
  text-align: justify;
}
