การตั้งค่า .gitignore สำหรับทุก git directory ในเครื่อง

Sep 27, 2024

บันทึกไว้ว่าในเครื่องมี gitignore อะไรบ้าง

คำสั่งที่ใช้

# สั่งให้ใช้ไฟล์ ~/.gitignore เป็น reference
git config --global core.excludesFile '~/.gitignore'
